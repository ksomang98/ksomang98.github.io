프로젝트는 “진행 예정 → 진행 중 → 완료” 순서로 업데이트할 거야. 링크/데모는 준비되는 대로 여기에 붙이면 됨.

1) 운영·트러블슈팅 지원 챗봇 (RAG)

  • 목표: 장비/비전 시스템 문서를 기반으로 현장 질문에 근거 포함 답변 제공
  • 핵심: 문서 업로드 → 질문 → 답변 + 근거(섹션/페이지) 표시
  • 진행: 준비 중
  • Repo: 추가 예정
  • Demo: 추가 예정

2) 불량 검사 + 실패 케이스 분석 + 엣지 추론 가정

  • 목표: 전처리 → 검사(규칙+AI) → OK/NG → 리포트 저장
  • 실전: 조명/각도/초점 변화로 실패 케이스 생성, 오검출/미검출 분석
  • 엣지: FPS 측정, 해상도/모델 경량화 시도, trade-off 정리
  • 진행: 준비 중
  • Repo: 추가 예정
  • Demo: 추가 예정
